Determine Ability Scores

PHB 12

You can do it randomly, with a set of numbers or buying points

Random method

Roll 6d6 and sum the 3 highest for each ability

Set of numbers

Use the following numbers 15, 14, 13, 12, 10,8

Buy points

You have 27 points to spend. Score/cost are: 8/0, 9/1, 19/2, 11/3, 12/4, 13/5, 14/6, 15/7

Finish

  • Saving Throws: add your ability modifier and for the checked ones add the PB
  • Skills: if normal add the ability modifier, if proficient add also the PB, if expert add 2*PB
  • Armor Class: default 10 + DEX + armor
